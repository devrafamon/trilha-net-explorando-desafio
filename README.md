# 📚 Sobre o Projeto

Este projeto foi desenvolvido como um desafio prático da trilha **Explorando a Linguagem C#**.

A proposta do desafio é permitir que desenvolvedores pratiquem conceitos importantes da linguagem enquanto constroem uma aplicação funcional utilizando **C# em um projeto .NET**.

Desafios desse tipo ajudam a transformar o aprendizado teórico em **experiência prática de programação**, fortalecendo habilidades essenciais para desenvolvimento backend. ([DIO][1])

---

# 🎯 Objetivo do Desafio

O principal objetivo deste projeto é aplicar conceitos da linguagem C# em um cenário prático, incluindo:

* estruturação de código em projetos .NET
* organização de classes e métodos
* aplicação de lógica de programação
* manipulação de dados
* interação com o usuário via console

O desafio serve como um exercício de consolidação das habilidades adquiridas durante a trilha.

---

# ⚙️ Funcionalidades Implementadas

O programa executa uma série de operações baseadas em lógica de programação e manipulação de dados.

Entre as principais funcionalidades estão:

* leitura de dados fornecidos pelo usuário
* processamento de informações utilizando lógica de programação
* organização de funcionalidades em classes e métodos
* exibição de resultados no console

A aplicação demonstra como estruturar uma **aplicação simples em C# utilizando o framework .NET**.

# 🧠 Conceitos Aplicados

Durante o desenvolvimento do projeto foram explorados conceitos importantes da linguagem C#.

### 📦 Programação Orientada a Objetos

Estruturação do código utilizando classes e métodos para representar regras de negócio.

### 🔄 Estruturas de Controle

Uso de estruturas como:

* `if` / `else`
* `switch`
* `for`

Essas estruturas permitem controlar o fluxo da aplicação.

---

### 📋 Manipulação de Dados

Utilização de estruturas de dados e variáveis para armazenar e processar informações.

Interação com o usuário através de entrada e saída de dados utilizando o terminal.
